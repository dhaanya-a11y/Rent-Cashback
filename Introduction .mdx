# Rent Cashback Offer Documentation

Welcome to the **Rent Cashback Offer** documentation.

This guide explains the complete configuration of the **Rent Cashback** transactional offer in the Offer Management System (OMS). It covers every stage of the offer lifecycle, including offer creation, customer eligibility, transaction qualification, cashback calculation, usage limits, posting rules, and customer-facing UI configuration in the Progressive Web App (PWA).

By following this documentation, users can understand how the offer is configured, managed, and presented to customers while ensuring accurate cashback processing and a seamless customer experience.
