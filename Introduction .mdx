# Welcome to Customer Value Proposition (CVP) Documentation

Welcome to the **Customer Value Proposition (CVP)** documentation. This repository provides implementation guides for configuring customer offers in the **Offer Management System (OMS)** and understanding how they are presented in the **Progressive Web App (PWA)**.

The documentation is intended to help product managers, implementation teams, QA engineers, and business users configure offers accurately while ensuring a consistent customer experience.

---

# What You'll Find

Each offer guide includes a complete walkthrough of the offer configuration process, covering:

- Offer Overview
- Business Objective
- Basic Details
- Offer User Base
- Transaction Eligibility
- Offer Calculation
- Usage Limits
- Offer Posting
- Customer-Facing UI Configuration
- Terms & Conditions
- Summary
